# AI-assistant-website

This repository has been converted into a web game arcade called PlayHub.

## What is included

- `index.html` — main game platform with clean navigation and game launch interface
- `styles.css` — sleek dark arcade theme with polished cards and responsive layout
- `script.js` — five playable browser games with live interaction

## Games included

1. Tic Tac Toe
2. Memory Match
3. Snake
4. Minesweeper
5. Trivia Quiz

## How to run

1. Open `index.html` in your browser.
2. Pick a game from the hub or top navigation.
3. Toggle between `Human` and `AI` mode if you want computer-driven gameplay.
4. Use the `Reset Current Game` and `Return to Hub` buttons as needed.

## Notes

- This is a static HTML/CSS/JavaScript arcade platform.
- Each game includes animated transitions and gameplay feedback.
- AI mode enables automatic play where available, including Tic Tac Toe, Memory Match, Snake, Minesweeper, and Trivia Quiz.

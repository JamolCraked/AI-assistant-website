# AI-assistant-website

This repository contains a class project page for an AI assistant interface.

## What is included

- `index.html` — the main page with a modern AI assistant layout
- `styles.css` — gradient background, cards, responsive chat UI
- `script.js` — interactive chat behavior with sample AI-style replies

## How to run

1. Open `index.html` in your browser.
2. Type a message in the chat input.
3. Click `Send` or use one of the suggested prompts.

## Notes

- This is a static HTML/CSS/JavaScript project.
- You can customize the sample responses in `script.js`.
- Use the `Start Chatting` and `Reset Conversation` buttons to interact with the page.
